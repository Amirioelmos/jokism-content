# Jokism Content

سرویس Go برای برداشتن شورتزهای پربازدید از بخش Explore آپارات و ارسال دوره‌ای آن‌ها به کانال بله با چند worker.

## راه‌اندازی

1. در بله یک ربات بسازید، آن را مدیر کانال کنید و اجازه ارسال رسانه بدهید.
2. فایل تنظیمات محیطی را بسازید:

   ```bash
   cp .env.example .env
   ```

3. `BALE_BOT_TOKEN`، `ADMIN_PASSWORD` و در صورت نیاز رمزهای Postgres را در `.env` پر کنید و اجرا کنید:

   ```bash
   docker compose up -d --build
   ```

4. پنل را در `http://localhost:8080` باز کنید. نام کاربری پیش‌فرض `admin` است. فاصله هر نوبت، تعداد شورتز در هر نوبت، تعداد worker، شناسه کانال، لینک کانال برای کپشن و سقف حجم را ذخیره کنید.

برای اجرای مستقیم: `BALE_BOT_TOKEN=... ADMIN_PASSWORD=... go run ./cmd/server`.

## رفتار سرویس

- در هر نوبت به صورت پیش‌فرض ۶ شورتز ارسال‌نشده از `shorts-explore` آپارات، مرتب‌شده بر اساس بیشترین بازدید، وارد صف می‌شود.
- شناسه ویدیوهای موفق در PostgreSQL و جدول `sent_videos` ثبت می‌شود؛ کلید اصلی جدول اجازه ارسال تکراری نمی‌دهد.
- ارسال اصلی با URL مستقیم فایل آپارات انجام می‌شود؛ پس سرویس معمولاً نیازی به دانلود و آپلود دوباره ویدیو ندارد.
- قبل از ارسال، حجم فایل با `Content-Length` چک می‌شود و اگر ویدیو بزرگ باشد یا بله خطای `413` بدهد، آن شورتز skip و ثبت می‌شود تا سرویس روی یک آیتم گیر نکند.
- کپشن از فایل `captions.txt` به صورت تصادفی انتخاب می‌شود و مقدار `BALE_CHANNEL_LINK`، مثل `@jokism_clip`، در انتهای آن می‌آید.
- موزیک‌ها به صورت پیش‌فرض از API سایت iTunes.ir خوانده می‌شوند؛ اول کاور آهنگ با عنوان خواننده/آهنگ و تگ `@jokism_music` در یک پست جدا ارسال می‌شود، سپس خود موزیک با کپشن کوتاه `@jokism_music` ارسال می‌شود. برای برگشت به منبع‌های قبلی می‌توانید `MUSIC_PROVIDER=iraniandj`، `MUSIC_PROVIDER=radiojavan` یا `MUSIC_PROVIDER=musics-fa` بگذارید.
- endpoint عمومی `GET /healthz` برای health check وجود دارد؛ پنل با Basic Auth محافظت می‌شود.
- توکن ربات فقط از environment خوانده می‌شود و داخل فایل تنظیمات یا HTML قرار نمی‌گیرد.

## نکته حقوقی و عملیاتی

تنها محتوایی را بازنشر کنید که اجازه‌اش را دارید و قوانین/حق نشر آپارات و بله را رعایت کنید. endpoint عمومی شورتز آپارات ممکن است تغییر کند؛ به همین دلیل `APARAT_SHORTS_API_BASE` قابل تنظیم است و خطا در پنل ثبت می‌شود.
